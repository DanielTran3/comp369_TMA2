Welcome To SiteMark This application is a short course which will teach you about some of the basics for the fundamentals required to develop a web application! There will be discussion about the following topics: HTML, CSS, XML, and Ajax. These topics are vital in developing a web application. These topics are separated into 3 units, where it is recommended that the units are completed in order. There will also be quizzes at the end of each chapter, so pay close attention to the material! Have fun and learn lots!!!
